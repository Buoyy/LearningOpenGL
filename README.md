## Description
Currently there's 10 textured cubes rotating randomly.
You can *move around* the scene with **arrow keys or WASD**, *look around* with **mouse** and *move up and down* with **space and control** respectively.
---
Note that the mouse controls are ***not*** inverted.
To do so, go to ***src/main.c*** and change the `mouse_invert` variable to true.
